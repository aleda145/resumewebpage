Home Server

I am currently running a server at home, which dutifully sent this website to you. It is running debian with a bunch of docker containers. I am using traefik as my reverse proxy. Traefik routes my subdomains to exposed docker containers and will automatically enable https on them too. This makes it really easy to spin up new services! It is as simple as just defining a label for which address the docker container should listen to. Elastic

I have an elastic stack running on docker, as seen in the picture above. I am using the normal elastic stack, so Elasticsearch, Logstash, Kibana and Beats(Filebeat and Metricbeat). I send my authentication logs from the debian server containing all the addresses of hackers trying to hack my server to Logstash. Logstash then processes it for Elasticsearch, which is then sent to a dashboard in Kibana, that can be viewed here!
